# Building and Running the Class Attendance Backend

This document provides instructions for setting up, building, and running the Class Attendance backend application.

## Prerequisites

- Node.js (v18 or later)
- npm (v10 or later)
- PostgreSQL (v14 or later)

## Dependencies Installation

Clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd class-attendance-backend
npm install
